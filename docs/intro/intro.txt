==============================
coords-transformer at a glance
==============================

coords-transformer is a library to transform pixel coordinates to latitude and longitude, and vise versa.
It's useful when you'd like to give latitude and longitude to a certain point within in a image.

Install coords-transformer
==========================

You can install this library via pip.

.. parsed-literal::

	pip install coords-transformer

Verifying
=========

To verify that coords-transformer can be seen by Python, type ``python`` from your shell.
Then at the Python prompt, try to import coords-transformer:

.. parsed-literal::

    import coords-transformer
    print(coords-transformer.get_version())

You may have another version of coords-transformer installed.
